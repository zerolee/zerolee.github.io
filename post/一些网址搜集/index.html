iter2

iter2 是一个对自带的 generator 完全兼容的实现，它提供用 iter2-defun 和 iter2-lambda
替代自带的 iter-defun 和 iter-lambda。
至于其他的函数或者宏(如： iter-yield, iter-next) 用自带的就好了。

优点:

  • 支持 save-excursion 以及类似的 similar special forms 以及宏 save-match-data。

  • 生成器函数可以使用 Edebug 进行调试。

  • 更快地转换复杂的生成器函数。

  • 通常更快的结果函数。

  • 生成的代码要小得多，尤其是对于复杂的函数。

  • 更具可读性的结果函数和回溯。

  • 对生成的迭代器函数的内置跟踪支持。

缺点

  • 因为iter2转换被高度优化，它不像原始generator包那样通用，因此更容易出现错误。

Async/Await for Emacs

这是受 TypeScript 的 Async/Await 实现启发的的一个简单实现。

依赖于自带的 generator.el 和 第三方 promise.el、 iter2
详细教程，请参阅 What about Async/Await? (TypeScript)

Promises/A+ for Emacs

这是Promises/A+的简单实现，移植自 https://github.com/then/promise

  • 可以使用与 JavaScript 版本 Promise 相同的 API。
  • then，catch、resolve、reject、all、race 等等……
  • 支持 “thenable”
  • 支持 “Inheritance of Promise”
  • 支持 “rejection-tracking”

详细教程，请参阅: www.promisejs.org (JavaScript).

Promises/A+ for Emacs 被用于 Async/Await for Emacs, 因此您可以使用 Async/Await 进行异步编程。.
